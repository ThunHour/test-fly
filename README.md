# Ecommerse Web-API

### Requirement
**Node Version >= 16**

### Set Up

```
yarn install
```
 
### Run Local Development

```
yarn dev
```

### Sync Database (Migrate + Generate + Push)

```
yarn db:sync
```

### Seed Database

```
yarn db:seed
```

### Open Prisma Studio

```
yarn db:studio
```

Set env (**DATABASE_URL**) -> postgres://postgres:password@localhost:5432/databasename?schema=public
